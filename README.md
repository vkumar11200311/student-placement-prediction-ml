
# 🎓 Student Placement Prediction Using Machine Learning

This project predicts whether a Computer Science student is likely to be placed based on academic performance and skill-related features.

## Features

- IQ Score
- CGPA
- 10th Marks
- 12th Marks
- Technical Skills
- Communication Skills

## Machine Learning Model

The project uses:

- Random Forest Classifier
- StandardScaler
- Python
- Pandas
- Scikit-learn
- Streamlit

## Project Workflow

Student Data
↓
Data Preprocessing
↓
Feature Scaling
↓
Random Forest Model
↓
Placement Prediction

## Output

The application predicts:

- Placed
- Not Placed
- Placement Probability

## Disclaimer

This project uses a synthetic dataset and is intended for educational purposes only. The predictions should not be used for real-world placement decisions.
