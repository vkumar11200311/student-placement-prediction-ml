An interactive Machine Learning web application built with Streamlit that predicts whether a Computer Science student is likely to be placed based on academic performance and skill-related attributes.

The application also provides an estimated placement probability generated by the trained classification model.

---

# 📌 Project Overview

**Student Placement Prediction System** is an end-to-end Machine Learning project that demonstrates the complete lifecycle of a classification model — from data analysis and preprocessing to model training, evaluation, serialization, and cloud deployment.

The system takes student information as input and predicts:

- 🟢 **Placed**
- 🔴 **Not Placed**

It also provides an estimated **Placement Probability** to make the prediction more informative.

> ⚠️ **Educational Disclaimer:** This project uses a synthetic dataset and is developed for educational and portfolio purposes only. The predictions should not be used to make real-world decisions regarding student employability or placement.

---

# 🎯 Project Objectives

The main objectives of this project are:

- Build a binary classification model for student placement prediction.
- Analyze the relationship between academic performance and placement outcomes.
- Explore the impact of technical and communication skills.
- Compare different Machine Learning classification algorithms.
- Evaluate models using standard classification metrics.
- Save the trained Machine Learning model for deployment.
- Build an interactive web application using Streamlit.
- Deploy the application on Streamlit Community Cloud.

# 📊 Features Used

The model uses the following student attributes:

| Feature | Description | Range |
|---|---|---|
| 🧠 IQ | Student IQ Score | 70–145 |
| 🎓 CGPA | Cumulative Grade Point Average | 4.0–10.0 |
| 📘 10th Marks | Percentage obtained in 10th grade | 40–100% |
| 📗 12th Marks | Percentage obtained in 12th grade | 40–100% |
| 💻 Technical Skills | Technical skill rating | 1–10 |
| 🗣️ Communication Skills | Communication skill rating | 1–10 |
